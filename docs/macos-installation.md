# obi-one for macOS

この配布物は Intel Mac と Apple Silicon Mac（M1/M2/M3/M4）に対応する universal build です。

## インストール

1. DMG を開き、`obi-one.app` を `Applications` フォルダへドラッグします。
2. 初回起動時に macOS が開発元を確認できない旨を表示した場合は、「システム設定」→「プライバシーとセキュリティ」から `obi-one` の「このまま開く」を選択します。

DMG を利用できない場合は `obi-one-macos-universal.zip` を展開し、`obi-one.app` を `Applications` フォルダへ移動してください。

現在の配布物は Apple Developer ID による署名・notarization を行わず、ad-hoc 署名を使用しています。そのため、初回のみ上記の許可操作が必要です。
